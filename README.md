# trendywear-website
This project showcases a fashion website built to demonstrate SEO practices including keyword optimization, search visibility, blogging strategy, and digital marketing techniques to increase website traffic.
TrendyWear Fashion Website

This project is created for a Digital Marketing assignment. The website demonstrates how businesses can increase online presence using Search Engine Optimization (SEO), blogging content, social media promotion, and email marketing strategies.

Technologies Used:
HTML
CSS
JavaScript

Objectives:
Improve search engine visibility
Demonstrate keyword usage
Show content marketing through blogging
Drive traffic using digital marketing strategies
